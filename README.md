# AQI_Status_Prediction
📌 Project Overview
The AQI (Air Quality Index) Status Prediction System is a Machine Learning project that predicts air quality levels based on environmental and pollution parameters. The system analyzes historical air quality data and classifies the AQI status into categories such as Good, Satisfactory, Moderate, Poor, Very Poor, and Severe.
This project helps users understand air pollution trends and supports environmental monitoring and decision-making.

🎯 Objectives
Predict AQI status using machine learning algorithms.
Analyze air pollution data and identify important factors affecting air quality.
Provide accurate AQI classification for environmental awareness.
Visualize pollution trends and AQI distribution.

📊 Dataset
The dataset contains air quality parameters such as:
PM2.5
PM10
NO₂
SO₂
CO
O₃
NH₃
AQI Value
AQI Category
Dataset Source: Public Air Quality Dataset
